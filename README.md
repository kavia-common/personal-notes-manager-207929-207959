# personal-notes-manager-207929-207959

This project contains a React frontend for a personal notes manager.

- Frontend location: `frontend_react/`
- Dev server: http://localhost:3000

Quick start:
1. cd frontend_react
2. npm install
3. npm start

Environment variables (optional):
- REACT_APP_API_BASE or REACT_APP_BACKEND_URL to enable REST API mode (CRUD at /notes)
- REACT_APP_FEATURE_FLAGS for toggles like `autosave` and `search` (JSON or CSV)

See `frontend_react/README_NOTES_APP.md` for full details.
